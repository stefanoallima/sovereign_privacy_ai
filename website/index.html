Features

Privacy without compromise

Every feature is built around one principle: your sensitive data never leaves your machine unless you say so.

🔒

Local PII Encryption

ChaCha20-Poly1305 encryption for all sensitive data. Keys stored in your system credential manager, not in files.

🤖

Built-in Local LLM

Embedded Qwen3-8B runs entirely on your machine — no separate install, no background service. Zero cloud dependency for maximum privacy.

👥

Multi-Persona System

Tax Navigator, Psychologist, Career Coach — each persona has independent privacy and routing settings.

⚡

Hybrid Cloud Routing

When cloud AI is needed, only anonymized categorical attributes are sent. Real values stay local.

📄

Document Parsing

Ingest PDF and DOCX files with automatic PII detection. Sensitive fields are masked before processing.

🔄

Re-hydration Templates

Cloud generates responses with placeholders. Your local system fills in real values — PII never touches the network.

How It Works

The Privacy Pipeline

Your message goes through multiple privacy layers before any data reaches the cloud.

1
Your Message "I earn 62,500 and have a mortgage..."
2
Backend Router Decides: local only, cloud, or hybrid based on persona settings
3
Attribute Extraction Extracts categories: income 50k-75k, has_mortgage: true
4
Cloud LLM Receives only: "User in 50k-75k bracket, has mortgage, Box 1"
5
Re-hydration Fills [INCOME] placeholders with your real values locally
